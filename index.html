CSS basics

Published on October 25, 2018

In week 4 I was introduced to the basics of CSS. The basic Syntax of CSS consists of Selector { Declaration }. The declaration parts consists of
{ property:value }. For example:

p { font-family:Arial }

In this example p is the selector, font family is the property and Arial is the value.

I was also introduced to cascade rules and inheritance. The two rules of cascade are last rule – this is where if two selectors are identical with the latter taking precedence - and specificity – if one Selector is more than the other, that selector will take precedence. Inheritance allows you to give a style to the body selector which will also be applied to most child elements within your HTML. However, this doesn’t apply to all property types but it will apply to font-family and colour.

I was introduced to four types of selectors in CSS:

Type selector :
A selector that will target a specific element i.e. <p> <header> <h1>
Class selector :
Selector targets elements with the specific class tag.
ID selector :
A selector that targets an element with a specific id tag
Pseudo-class selector :
For certain elements such at the <a> tag for when the user interacts with the content it changes. Selector must have a style for a link, a visited link, a hover link and an active link. It can also have a focus link for when there is keyboard focus.

What I have achieved in my lab tutorial:

  • I saved the normalize.css and my stylesheet.ccs files onto my W: drive.
  • I used type selectors to style different elements and I understood how style precedent works.
  • I used class and ID attribute to style specific elements
  • I used pseudo-class selectors to style my Hyperlinks and I also used a compound selector to style my nav tag with the class attribute menu.
  • I experimented with different style attributes in CSS.

Flex-box model

Published on November 22, 2018

In week 8 I styled my tutorial page using the flexbox model. By giving an element the declaration – {display:flex} - the contents within that element will be wrapped in a ‘flex-container’. This makes it easier to layout the page in a particular direction, control the dimensions of the ‘container’ and the spacing between each ‘container’.

Each step on my tutorial page was wrapped in a container called ‘card’. I styled each ‘card’ the same and I was able to achieve the desired layout on my tutorial page. I believe this gives my tutorial page a clean and modern look.

I also used the position property to credit the photos I have used. To do this I had to give the parent element ‘card’ the position property value of relative and the child element ‘copy’ property value absolute. I could then position the child element using properties bottom and right.
